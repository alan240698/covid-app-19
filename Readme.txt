1. Tạo project reactjs trong thư mục hiện tại
->npx create-react-app .

2. Cài đặt các thư viện cần thiết cho project
- Tương tác với api:
npm install --save axios
- Hiển thị lên các biểu đồ
npm install --save highcharts
- Libary giúp sử dụng highcharts trong react
npm install --save highcharts-react-official
- Thao tác nhanh các hàm trong javascript
npm install --save lodash
- Các định dạng về ngày giờ
npm install --save moment
- Lấy các dữ liệu từ bảng đồ
npm install --save @highcharts/map-collection
- Giao diện ui material 
npm install --save @material-ui/core
- Các font chữ 
npm install --save @fontsource/roboto


=> Lệnh cài tổng hợp những cái trên
-> npm install --save axios highcharts highcharts-react-official lodash moment @highcharts/map-collection @material-ui/core @fontsource/roboto

3. Xóa các file không cần thiết
